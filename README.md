# Stock Trader – User-Defined Algorithmic Trading Platform

A cloud-based system that allows users to upload custom Python trading
strategies executed against the Alpaca paper trading API. Execution
results are stored in Firebase Firestore and visualized using FlutterFlow.

## Tech Stack
- Python (Flask)
- Firebase (Auth, Storage, Firestore)
- Alpaca Trading API (Paper Trading)
- FlutterFlow

## Key Concepts
- Server-side execution of user-defined logic
- API integration and cloud storage workflows
- Frontend–backend decoupling

## Disclaimer
This project is for educational purposes only.
No real trades are executed.
